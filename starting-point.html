Starting point

Imagine that your company works on a greeter library, and the current code looks as follows:

class Greeting
  def print()

    puts "Hello!"
  end
end

Feature request

The new feature request comes in, and you are responsible for delivering it. The requirement is: “Add support for custom greeting string”. Since the company uses ticketing system, it also has a reference: GRTR-5224

You start by creating a new branch:

git checkout -b grtr-5224-add-support-for-custom-greeting-string

Rules

  • Name branches descriptively. If you decide to follow the format above, the script can be created to fetch the ticket title and generate the branch name for you.
